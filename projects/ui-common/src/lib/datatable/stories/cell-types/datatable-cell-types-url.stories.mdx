## Url

> Url cell type is not implemented yet and will just display as a string for now.

### Config Model

```
{
  type = 'url',

  /**
   * Styles added to the root cell elements `style` attribute.
   */
  styles?: 'string' | 'string'[]

  /**
   * Classes added to the root cell elements `class` attribute.
   */
  cssClass?: 'string' | 'string'[]
}
```

### Definition

#### No Config

```
{
  ...,
  cellype = 'url'
}
```

#### Basic Config
```
{
  ...,
  cellType = {
    type = 'url'
  }
}
```

#### Config with styles
```
{
  ...,
  cellType = {
    type = 'url',
    styles = 'color: red; background-color: blue'
  }
}
```

## String

> 'string' is the default cell type, so you can leave the cell type undefined unless config is needed.

### Config Model

```js
{
  type = 'string',

  /**
   * Styles added to the root cell elements `style` attribute.
   */
  styles?: string | string[]

  /**
   * Classes added to the root cell elements `class` attribute.
   */
  cssClass?: string | string[]
}
```

### Definition

#### No Config

```js
{
  ...,
  cellype = 'string'
}
```

#### Basic Config

```js
{
  ...,
  cellType = {
    type = 'string'
  }
}
```

#### Config with styles

```js
{
  ...,
  cellType = {
    type = 'string',
    styles = 'color: red; background-color: blue'
  }
}
```

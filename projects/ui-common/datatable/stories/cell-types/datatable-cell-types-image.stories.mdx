## Image

> Image cell type is not implemented yet and will just use `icon` for now.

### Config Model

```js
{
  type = 'image',

  ...
  // Refer to 'icon' model for now.
}
```

### Definition

#### No Config

```js
{
  ...,
  cellype = 'image'
}
```

#### Basic Config

```js
{
  ...,
  cellType = {
    type = 'image'
  }
}
```

#### Config with styles

```js
{
  ...,
  cellType = {
    type = 'image',
    styles = 'color: red; background-color: blue'
  }
}
```
